<h2>Links and references</h2>



<h3>Software using FLINT</h3>

<ul>
<li><a href="http://sagemath.org">Sage</a> uses FLINT as the default package for polynomial arithmetic over Z, Q and Z/nZ for small n.</li>
<li>Work is currently in progress to use FLINT in <a href="http://www.singular.uni-kl.de">Singular</a> and <a href="http://www.math.uiuc.edu/Macaulay2/">Macaulay2</a>.</li>
<li><a href="http://www.hcrypt.com/scarab-library/">Scarab library</a> - an implementation of fully homomorphic encryption using FLINT</a>.</li>
</ul>
